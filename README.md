הטסת רחפן בתקשורת LoRa וקבלת נתוני GPS ו- RSSI
 
משתתפים: שמעון שור & שלמה בקר 

מנחה: ד"ר שמעון מזרחי 

מטרות הפרויקט: 
1. שליטה על הרחפן באמצעות ג'ויסטיק.
2. שידור וקליטה באמצעות LoRa
3. כתיבת קוד ל- Ma ו- Sl בשפת C
4. כתיבת קוד בשפת Pyhton לקריאת ושידור ערכי הג'ויסטיק והצגת נתוני ה-GPS.

רכיבי המערכת:
1. STM-32 ממשפחת NUCLEO
2. בקר Mbed sx1272 .2.1.5KK
3. בקר טיסה ג'ויסטיק G.P.S 10
4. בקר NUCLEO ARM -cortex M מעבד
